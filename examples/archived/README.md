# Archived Examples

This directory stores examples that were useful during earlier architecture transitions but no longer match the current Immunex command surface.

Typical reasons for archiving include:

- dependence on legacy pipeline factories
- dependence on pre-`imn` public APIs
- references to retired scripts or workflow entry points
- outdated RMSF and batch examples from earlier refactoring stages

Archived examples are kept for history and migration reference only.

If you need a current example, return to:

- `examples/README.md`
