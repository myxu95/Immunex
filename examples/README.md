# Examples

This directory only keeps examples that still match the current Immunex product surface.

## Recommended Starting Points

- `quality_analysis_usage.py`: trajectory quality assessment
- `docking_angles_usage.py`: docking-angle analysis
- `contact_correlation_example.py`: contact-correlation analysis
- `rmsd_with_index_manager.py`: RMSD usage under the current package layout
- `task_discovery_example.py`: task discovery and context generation

## Other Active Examples

The remaining files in this directory are still kept as working examples for specific submodules such as:

- chain standardization
- index generation and index management
- PDB download and standardization
- topology chain identification
- group selection and GRO copy helpers
- multimodel concatenation

## Archived Examples

Examples that rely on superseded workflows or legacy APIs have been moved to:

- `examples/archived/`

These files are kept only for historical reference and should not be used as the primary entry point for new work.
