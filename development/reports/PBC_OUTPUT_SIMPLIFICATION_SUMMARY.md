# PBC Pipeline Output Simplification

**Date**: 2026-03-17
**Modified Module**: `immunex/pipeline/pbc_rmsd_pipeline.py`
**Status**: ✅ Complete

---

## Problem Statement

**Before**: Pipeline generated too many output files (5-6 files)
- Multiple report formats (JSON, Markdown, TXT)
- Redundant information
- No visualization
- Difficult to quickly understand results

**User Requirements**:
1. Simplify output - only essential files
2. Add RMSD plot for quick visual assessment

---

## Solution

### Output Simplification

**BEFORE** (5 files):
```
output/{system_id}/
├── md_pbc.xtc                  # 614 MB - PBC-corrected trajectory
├── md_convergence.txt          # 783 B  - Convergence analysis
├── md_quality_metrics.json     # 43 KB  - Technical metrics (JSON)
├── md_quality_report.md        # 820 B  - Quality report (Markdown)
└── rmsd.csv                    # 27 KB  - RMSD data
```

**AFTER** (4 files):
```
output/{system_id}/
├── md_pbc.xtc                  # 614 MB - PBC-corrected trajectory
├── md_quality_report.txt       # <1 KB  - Unified quality + convergence report
├── rmsd.csv                    # 27 KB  - RMSD data
└── rmsd.png                    # 347 KB - RMSD plot with statistics (NEW!)
```

**Cache files** (auto-generated by MDAnalysis, hidden):
- `.md_pbc.xtc_offsets.npz` - Trajectory offset cache
- `.md_pbc.xtc_offsets.lock` - Concurrency lock

---

## Changes Made

### 1. Modified `_generate_reports()` Method

**Removed**:
- JSON metrics file generation (`md_quality_metrics.json`)
- Markdown report generation (`md_quality_report.md`)
- Separate convergence report (`md_convergence.txt`)

**Added**:
- Unified TXT report (combines quality + convergence)
- RMSD plot generation (PNG with statistics overlay)

### 2. Added `_generate_unified_text_report()` Method

Combines all quality information into single TXT file:
- Overall grade and status
- RMSD statistics (mean, std, min, max)
- Convergence analysis (converged status, convergence time)
- Block analysis (5 blocks)
- Moving average analysis (drift detection)
- Post-PBC validation (COM drift, Rg stability)

### 3. Added RMSD Plotting

Auto-generates `rmsd.png` with:
- Time series plot (blue line)
- Mean RMSD line (red dashed)
- Statistics box (Grade, Mean±Std, Converged status)
- High-resolution (300 DPI)

### 4. Removed Obsolete Methods

Deleted unused methods:
- `_generate_markdown_report()`
- `_serialize_results()`
- `_serialize_dict()`

### 5. Added PlotManager Integration

Import `PlotManager` for plotting functionality.
