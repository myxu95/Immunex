# Question Patterns

## Pattern: top residue
Examples:
- Which residues are the most important?
- Which residue is the strongest hotspot?

Expected answer shape:
- one direct answer sentence
- 3 evidence bullets
- optional top 3 to top 5 ranked residues

## Pattern: top pair
Examples:
- Which pair is strongest?
- Which persistent pair is most stable?

Expected answer shape:
- one direct answer sentence
- 3 evidence bullets
- optional top 3 ranked pairs

## Pattern: top region
Examples:
- Which region is most active?
- Which interface region dominates the system?

Expected answer shape:
- one direct answer sentence
- 3 evidence bullets
- optional top 3 regions

## Pattern: cluster status
Examples:
- Which cluster is dominant?
- Is this a single-state or multi-state system?

Expected answer shape:
- one direct answer sentence
- 3 evidence bullets
- dominant cluster and population share

## Pattern: quality status
Examples:
- Is the trajectory stable?
- Does the system look well-behaved?

Expected answer shape:
- one direct answer sentence
- 3 evidence bullets
- explicit stable / caution / unstable wording
